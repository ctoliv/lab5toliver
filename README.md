# Lab 5 - Custom User Bitmap

This repository contains my Lab 5 project for CPSC 440 Game Programming.

## Description

This lab creates a custom user bitmap using Allegro primitives. The bitmap is 64 by 64 pixels and is designed as a small ship/drone object.

The bitmap uses multiple primitive shapes and colors. Four versions are created to show the object facing different directions: up, right, down, and left.

## Features

- Creates a 64x64 bitmap
- Uses at least four Allegro primitives
- Uses at least three colors
- Draws a custom ship/drone design
- Creates four directional versions of the bitmap
- Displays all four bitmaps on the screen
- Escape key closes the program

## Primitives Used

- Filled rectangle
- Filled circle
- Filled triangles

## How to Run

1. Open the `.sln` file in Visual Studio.
2. Make sure Allegro is installed through NuGet.
3. Make sure the Primitives Addon is enabled.
4. Build and run the project.
5. Press Escape to close the program.
